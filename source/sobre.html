Trabalho 03

Ambiente operacional do servidor multiplayer

O projeto simula a administração de um servidor Linux usado para disponibilizar um site relacionado ao Pong Multiplayer.

Container Ubuntu

Responsável por simular a instância Linux do cenário cloud.

Apache

Responsável pela publicação das páginas estáticas no navegador.

Shell Script

Responsável pela automação das tarefas operacionais do ambiente.

Fluxo simplificado

Navegador → localhost:8080 → Container Ubuntu → Apache → Site Pong Online
